# Random Quote Generator (Генератор случайных цитат)

## Автор
Имя и Фамилия: [Семен Чекалдин]

## Описание
Графическое приложение на Python (Tkinter), которое:
- Генерирует случайные цитаты из встроенной коллекции.
- Ведёт историю сгенерированных цитат.
- Позволяет фильтровать историю по автору или теме.
- Поддерживает ручное добавление новых цитат с валидацией ввода.
- Сохраняет и загружает историю в формате JSON.

## Зависимости
Только стандартные библиотеки Python:
- `tkinter`
- `random`
- `json`
- `os`
